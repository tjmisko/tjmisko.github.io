* **What it is.** A freelance job turning the Pomera DM250 — a small ARM-based writing gadget — into a dual-boot device running a custom, minimal Debian Linux setup with Chinese (Pinyin) input and a stripped-down writing interface.
* **Why it matters.** It gave the client a focused, distraction-free writing machine on open software, with capabilities — a real Linux environment, Chinese input — the stock device never shipped.

**Technical details**

* Installed and configured armhf Debian Linux as a dual-boot OS on the Pomera DM250, an embedded ARM writing device
* Enabled Chinese language input via Pinyin and ibus, with extensibility for additional input methods
* Designed a minimalist writing UI using i3, xfce4-terminal, and custom bash scripts to meet client specifications
