* Installed and configured armhf Debian Linux as a dual-boot OS on the Pomera DM250, an embedded ARM writing device
* Enabled Chinese language input via Pinyin and ibus, with extensibility for additional input methods
* Designed a minimalist writing UI using i3, xfce4-terminal, and custom bash scripts to meet client specifications
